Export scene file — 3ds Max → Blender

Отдельная команда меню (не режим Render). Две цели: Blender scene (.blend) — по умолчанию: DCC уходит на сервер, тот собирает .blend и возвращает его как результат (на рендеринг не отправляет) — фактически конвертер 3ds Max → Blender, самостоятельная ценность. Либо чистый DCC JSON — сырой нейтральный экспорт локально. Рендера тут нет.

Две цели формата
Blender scene (.blend)default
DCC → сервер → .blend назад как результат. Конвертация 3ds Max → Blender (своя ценность). На рендеринг не уходит.
server round-trip
DCC JSON (.dccjson)
Сырой нейтральный экспорт сцены. Для пайплайна, отладки, архива.
локально · без round-trip

Дефолт — Blender, потому что готовый .blend ценен сам по себе (конвертация сцены, а не только промежуточный формат). Pack textures & assets вшивает зависимости (иначе пути остаются ссылками). Тот же DCC-пайплайн, что внутренний submit в Render, но результат — файл у пользователя, без отправки на ферму.

Гейт: плагин — только для залогиненного. Export…, как и Render…, disabled до логина — диалог открывается уже с сессией (аккаунт всегда в баре). Разница режимов: Blender — server round-trip (DCC → сервер → .blend); DCC JSON — локально, без round-trip.
